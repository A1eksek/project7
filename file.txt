Фабричный метод - это пораждающий патерн проектирования который определяет общий интерфейс
для создания объектов в супер классе позваляет подклассам изменять тип создавания объектов

Прототип - патерн пораждающие объект, задаёт виды с помощью экземпляра вида и создаёт
новый объекты путём копирования этого прототипа

Посрденик - это поведенченский патерн пректирования который позваляет уменьшить связанность множества классом между собой
благодоря перемещению этих связий класс посредник
